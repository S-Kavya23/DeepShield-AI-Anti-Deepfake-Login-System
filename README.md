## Overview
This project is created using AI tools. The application was developed by providing prompts to AI models, which assisted in generating the core code, functionality, and structure. This README explains the purpose, usage, and features of the project.

## Features
- AI powered solution
- Liveness detection
  - Face Detection
  - Voice Verification
  - Fingerprint Authentication]
  
  ## How It Works

1 Choose Your Authentication Mode
Select between Full Secure Login or Camera + Voice Only based on your security needs.

2 Complete Verification Steps
Follow the guided process for face detection, voice verification, and fingerprint authentication.

3 Secure Access Granted
Once verified, enjoy secure access to your account with the peace of mind that your identity is protected.
